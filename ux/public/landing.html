Features

Everything your team needs

Built for engineers, designers, and managers who value speed and clarity.

✎
WYSIWYG Editor
Rich text editing with slash commands, tables, code blocks, and live formatting — no Markdown required.
◆
Mermaid Diagrams
Embed flowcharts, sequence diagrams, ERDs, and Gantt charts directly in any page.
👤
Real-time Collaboration
See collaborators' cursors live. Edit together without conflicts using operational transforms.
🔍
Semantic Search
Vector-powered full-text search that understands meaning, not just keywords.
📄
PDF Export
Export any page or section to a clean, print-ready PDF with one click.
🔗
MCP API
Model Context Protocol integration — let AI assistants read and write your wiki natively.
🔔
Webhooks
Trigger external systems on page create, update, or delete events. Integrates with Slack, Teams, and more.
📱
PWA
Install CloudWiki as a desktop or mobile app. Works offline with service worker caching.
